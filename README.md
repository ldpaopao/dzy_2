## 📚 项目简介
这是一个基于 Spring Boot + Vue3 的抽奖系统，支持注册、登录、抽奖、中奖信息查看、后台管理等功能。

## 🚀 技术栈
- 前端：Vue3、Element Plus、Axios、Vite
- 后端：Spring Boot、MyBatis、RabbitMQ
- 数据库：MySQL
- 注册中心：Nacos

## 🔧 功能模块
- 用户注册与登录
- 实时抽奖与动画转盘
- 中奖信息记录
- 后台用户与奖品管理
